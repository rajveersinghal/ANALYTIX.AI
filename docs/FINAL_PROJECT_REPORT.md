## 4. Key Functional Modules

### A. The 11-Step AutoML Pipeline
The heart of the application is a synchronized pipeline that guides data through:
1.  **Upload**: Secure data ingestion.
2.  **Profiling**: Auto-quality scoring (0-100).
3.  **Understanding**: Deep univariate and bivariate analysis.
4.  **Cleaning**: Automated null handling and outlier removal with Audit Logs.
5.  **EDA**: Strategic insight discovery.
6.  **Statistics**: Rigorous hypothesis testing.
7.  **Task Determination**: Domain-aware problem routing.
8.  **Modeling**: Hyperparameter optimization and model selection.
9.  **Explainability**: SHAP value computation and Global Feature Importance.
10. **Decision Support**: Strategic recommendation engine.
11. **Reporting**: One-click professional PDF/HTML exports.

### B. Intelligent Dashboards
- **Executive Dashboard**: A high-level overview of project health, key metrics, and model performance.
- **Sales Intelligence Module**: A specialized module deep-diving into sales trends, customer churn, and revenue forecasting with high-fidelity Recharts visualizations.

### C. Advanced AI Features
- **What-If Simulator**: An interactive environment where users can change variables (e.g., "What if price increases by 10%?") and see real-time predictive impacts.
- **Strategic Narrative Engine**: An AI layer that translates technical metrics (RMSE, R2) into human-readable strategic summaries.
- **Explainability (XAI)**: Integrated SHAP charts that show *why* a model made a specific prediction.
- **M5 AI Chat Widget**: A persistent, floating AI assistant that answers queries about the data and guides users through the platform.

## 5. Technical Architecture

### Frontend (Modern SPA)
*   **Framework**: React.js with Vite for blazing-fast development.
*   **Styling**: Tailwind CSS for responsive utility-first design.
*   **Animations**: Framer Motion for premium UI transitions.
*   **Visualizations**: Recharts and Plotly.js for interactive data charts.
*   **State Management**: Optimized React Context and Hooks for pipeline synchronization.

### Backend (Intelligence Layer)
*   **API Framework**: FastAPI (Python) for high-performance service delivery.
*   **Data Science Stack**: Pandas, Scikit-Learn, SHAP, Statsmodels.
*   **Pipeline Control**: A custom `MetadataManager` that ensures state persistence and resilience across sessions.
*   **Reporting Engine**: FPDF2 for dynamic, branded PDF generation.
